				Instructions to install and play the game:

Installation:
1. In the scrabble directory, run the command "make compile"
This will give an executable file "a.out". Run this file by writing "./a.out"

The game will start


Gameplay:
1. You will see a set of 7 letter below the board. Type a word formed by these letters and press Enter
2. Now input the number corresponding to the square you want to insert the starting letter of the word and press Enter.
3. Now type either 'h' or 'v' and press Enter for the word to appear horizontally/vertically respectively on the board.

For rules of the game scrabble refer - https://www.scrabblepages.com/scrabble/rules/

Enjoy the game!
